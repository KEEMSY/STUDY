# **Singleton 패턴**

하나의 클래스 타입에 대해서 오직 하나의 객체만이 생성되도록 보장해주는 패턴이다.

Singleton 패턴이 적용된 클래스의 객체는 다른 클래스들에서 접근할 수는 있지만 생성은 할 수 없다.
